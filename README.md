Jednostki w CSS

1. Cel projektu:

Jest to niewielka witryna internetowa opisująca jednostki używane w CSS.

2. Opis projektu:

Statyczna strona internetowa z 3 plikami .html oraz 2 plikami .css służąca jako proste kompedium informacji na temat jednostek w CSS oraz ich zastosowań. Brak aplikacj JS.

3. Struktura projektu:

/cite → Podstrony (.html)

/img → obrazy

/styles → pliki CSS

index.html → strona główna

README.md → plik README
